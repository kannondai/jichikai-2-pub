集会所の修繕費 — 会計のしかた（図解）

この資料では、この分担を帳簿にどう書くのか・会員にどう報告するのかを順番に説明します。結論として、報告はこの分担関係が見える形式が望ましく、両自治会の会計年度を揃えると処理がより簡明になります。
現在、第一自治会は1月末締め・第二自治会は2月末締めですが、第二自治会も1月末締めに合わせることが望ましいです。

ひと目でわかる要点

集会所の修繕費は、費用の性質ごとに合意した方法で両自治会が分担しています。

1. 集会所の共同利用のしくみ

graph LR A["第一自治会\n（224戸）"] --> C["🏠 集会所\n（共同利用）"] B["第二自治会\n（113戸）"] --> C style C fill:#f5f5dc,stroke:#888

集会所は両自治会が共同で管理しています。
修繕費が発生したとき、費用の種類によってどちらの自治会が先に支払うかが異なります。

  • 大規模修繕（屋根塗装など）：戸数の多い第一自治会が業者に支払い、後日第二自治会が負担分を第一自治会に支払います。
  • 小規模修繕（鍵修理など）：その都度の判断で、立て替えた側が後日精算します。

2. 帳簿の書き方（基本ルール）

よくある間違い ❌

「どうせ半分は返ってくるから、自分たちの負担分だけ書けばいい」

これは間違いです。会計は「実際に起きたこと」をそのまま記録するのが原則です。

正しい書き方 ✅

graph TD A["修繕費が発生"] --> B["全額を<br/>【支出】に計上"] A --> C["相手自治会から<br/>受け取った金額を<br/>【収入】に計上"] B --> D["支出 ー 収入<br/>＝ 実質負担額"] C --> D style B fill:#ffe0e0,stroke:#c00 style C fill:#e0ffe0,stroke:#080 style D fill:#e0e8ff,stroke:#339
区分 内容 考え方
支出 修繕費の全額 実際に支払った金額をそのまま書く
収入 相手自治会から受け取った負担金 受け取ったお金は収入として書く
実質負担 支出 ー 収入 これが自治会の実際の負担額

6. 会員への会計報告の書き方

総会で会員に報告する際は、以下の形式を参考にしてください。

報告例（屋根塗装修理・第一自治会の場合）

【集会所修繕費に関する会計報告】

項目 金額 備考
修繕費（屋根塗装） 430,000円 業者への支払い全額
うち 第二自治会負担分（受取） △144,000円 戸数比（224/337）による按分（1,000円単位に丸め）
当自治会（第一）の実質負担 286,000円 集会所積立金より支出

第二自治会との間で負担額の精算を完了しています。
分担の根拠：第一自治会224戸、第二自治会113戸の戸数比による按分。

報告例（鍵修理・第二自治会の場合）

【集会所修繕費に関する会計報告】

項目 金額 備考
修繕費（鍵修理） 11,000円 業者への支払い全額（立替）
うち 第一自治会負担分（受取） △5,500円 折半
当自治会（第二）の実質負担 5,500円 集会所積立金（特別会計）より支出

第一自治会との間で負担額の精算を完了しています。
分担の根拠：両自治会合意による折半。

報告のポイント：

  • 修繕費の全額と自治会の実質負担額を両方明記する
  • 分担の計算根拠（戸数比・折半など）を必ず書く
  • 精算が完了しているかどうかを明記する

7. 会計係のチェックリスト

修繕費が発生したとき、以下を順番に確認してください。

  • 領収書・請求書を受け取ったか
  • 第一自治会との分担方法（戸数比 or 折半）を確認・合意したか
  • 帳簿に全額を支出として計上したか
  • 第一自治会への請求を行ったか
  • 第一自治会からの入金を収入として記録したか
  • 一般会計・特別会計のどちらで処理するか確認したか
  • 両自治会の記録が一致しているか確認したか
  • 総会の会計報告に含めたか

8. よくある質問

Q. 返ってくる分があるなら、最初から自分たちの負担分だけ書いてはダメなの？
A. ダメです。後から監査が入ったとき、領収書（全額）と帳簿の金額が合わなくなります。「全額支出・分担金を収入」という形で記録することで、お金の流れが明確になります。

Q. 精算が翌年度になってしまった場合は？
A. 修繕した年度の支出として計上します。翌年度に精算したときは「前年度未収金の回収」として収入に記録します。費用の計上タイミングと精算のタイミングは別物です。

Q. 分担方法（戸数比か折半か）はどうやって決めるの？
A. 両自治会で話し合って決めます。大規模修繕は戸数比（世帯数に応じた公平な負担）、少額修繕は折半（簡便さ優先）が実績としてあります。どちらを使うかよりも、合意した根拠を記録に残すことが重要です。

9. 提言：両自治会の会計年度を揃えることが望ましい

現状の問題

現在、両自治会の会計年度末は以下のようにずれています。

会計年度末
第一自治会 1月末
第二自治会 2月末（会則上の正式な締め）

年度末がずれていると、「どの年度の費用・収入として記録するか」が自治会によって変わってしまい、記録の突き合わせが複雑になります。

解決策：会則改定で1月末に統一する

第二自治会の会計年度末を1月末に改定することで、このずれがなくなります。

メリット 説明
精算が同一年度内に完了しやすい 両会が同時期に締めるため、年度またぎが起きにくい
記録の突き合わせが簡単になる 決算のタイミングが揃い、確認作業がしやすくなる
会計係の誤解が減る 「年度外だから計上しない」という混乱の原因を構造的に解消できる

📌 手続きの注意：会計年度の変更は会則改定が必要です。総会での決議を経て、正式に変更してください。移行年度（2月末→1月末にする場合は11か月の短縮年度になる）の処理方法も事前に決めておくことが必要です。
